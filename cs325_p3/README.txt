runs on python version 3.10.11

i couldn't make a yaml file because i kept getting errors, but make sure that 'requests' is installed by typeing in the terminal: pip install requests.

run the MAIN file in a terminal by typeing: python main.py <URL>
where <URL>  is a reddit thread like 'https://www.reddit.com/r/funny/comments/16brnzb/self_aware/'.

puts the text of the website in data/raw/output.txt
puts the comments in data/processed/comments.txt
